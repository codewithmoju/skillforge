# App Color Scheme

## Core Palette
Defined in `app/globals.css`

### Primary
- **Primary**: `#3B82F6` (Blue 500)
- **Primary Glow**: `#60A5FA` (Blue 400)

### Accents
- **Cyan**: `#06B6D4` (Cyan 500)
- **Violet**: `#8B5CF6` (Violet 500)
- **Indigo**: `#6366F1` (Indigo 500)

### Backgrounds
- **Main Background**: `#0F172A` (Slate 900)
- **Card Background**: `#1E293B` (Slate 800)
- **Overlay**: `rgba(15, 23, 42, 0.8)`

### Text
- **Primary**: `#E2E8F0` (Slate 200)
- **Secondary**: `#94A3B8` (Slate 400)
- **Muted**: `#64748B` (Slate 500)

## Theme Specifics

### Cyber Neon
- **Grid**: Indigo (`#6366F1`) with transparency

### Forest Quest
- **Glow**: Green (`#22c55e`)

### Dragon's Lair
- **Base**: Red (`#ef4444`)

### Ocean Depths
- **Base**: Cyan (`#06b6d4`) / Teal (`#14b8a6`)

## Usage
These colors are available as CSS variables (e.g., `var(--color-primary)`) and Tailwind utility classes (e.g., `bg-primary`, `text-accent-cyan`).
